# Solicitação de Acesso a Sistemas

## Visão geral

Todo acesso a sistemas internos segue o princípio do menor privilégio: o colaborador recebe apenas os acessos necessários para a sua função.

## Como solicitar um novo acesso

1. Abra um chamado no portal de TI em suporte.empresa.com.br
2. Selecione a categoria "Solicitação de acesso"
3. Informe o sistema desejado e a justificativa
4. O chamado é encaminhado para o gestor aprovar
5. Após a aprovação, a equipe de TI provisiona o acesso em até 2 dias úteis

## Política de senhas

- Mínimo de 12 caracteres, com letras, números e símbolos
- Troca obrigatória a cada 90 dias
- É proibido compartilhar senhas entre colaboradores; cada pessoa usa seu próprio acesso
- O uso de gerenciador de senhas corporativo é recomendado

## Autenticação em dois fatores (2FA)

O 2FA é obrigatório para sistemas que tratam dados sensíveis (financeiro, RH e dados de clientes). A configuração é feita no primeiro acesso, via aplicativo autenticador.

## Desligamento

Quando um colaborador é desligado, o gestor deve abrir um chamado de revogação de acessos no mesmo dia. A TI desativa todos os acessos em até 4 horas.
